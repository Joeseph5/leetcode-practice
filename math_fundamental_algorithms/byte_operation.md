# 位操作
## 验证奇偶数
被验证数的二进制与'1'进行按位与操作，并这个数字的二进制最后一位，然后再进行判断。

```c++
// 奇数为1，偶数为0
(num & 1) == 1
```

## 数字交换

异或操作：相同数异或为0

```c++
x = (x ^ y);
y = x ^ y; // y = (x ^ y) ^ y = x ^ (y ^ y) = x ^ 0 = x
x = x ^ y;
```

## 集合操作

按位与求集合的并集，按位或求集合的并集。

假设我们有两个集合{1, 3, 8}和{4, 8}。我们先把这两个集合转为两个 8 位的二进制数，从右往左以 1 到 8依次编号，则为10000101和10001000。

按位与操作：10000000，交集为{8}

按位或操作：10001101，并集为{1, 3, 4, 8}
